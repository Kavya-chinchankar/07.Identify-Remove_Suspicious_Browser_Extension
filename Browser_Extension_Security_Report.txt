## üß© Additional Research: How Malicious Extensions Harm Users
- **Data Theft:** Can read keystrokes or access login credentials.  
- **Ad Injection:** Displays unwanted ads or redirects.  
- **Tracking:** Collects browsing history for sale or misuse.  
- **Browser Hijacking:** Changes homepage or search engine settings.  
- **Phishing:** Redirects to fake login pages to steal credentials.

---

## üß† Best Practices for Safe Extension Use
- Install only from trusted sources like the **Chrome Web Store** or **Mozilla Add-ons**.  
- Always **check permissions** before installation.  
- Read **user reviews and ratings**.  
- **Limit the number of installed extensions**.  
- Keep extensions **updated regularly**.  
- Remove unused or suspicious extensions immediately.
